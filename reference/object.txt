  Paramater list and basic object manipulating commands are demonstrated by me here.

Syntax
------
N.B. Coordinates are in PIXELs.  There are 64 pixels to one side of each map cube.

Object:
200 1 (12345,3456,256) FUTURE 0 512


Creation
--------
100 MAKEOBJ 200 0 0 0 0

Removal:
102 KILLOBJ 200 0 0 0 0
 OR
Leave out the '1' flag from the FUTURE line and use:
102 RESET 0 0 -1 -1 0 {clears entire mission environment}
