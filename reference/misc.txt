SETBOMB	<car> <succ> <fail> <bomb> <score>
Sets the car bomb value of <car> to <bomb>, some useful values of bomb are
	0 - No Bomb (disarm)
	1 - Unarmed ‘5 second’ bomb
	2 - Armed ‘5 second’ bomb - i.e. it will blow in five seconds
	3 - Model Car Bomb (instant detonation on pressing ‘special key’)
	4 - Dodgy Explosives (take some damage then it will blow)
	5 - Unarmed Speed Bomb (Speed >ľ max speed to arm)
	6 - Armed Speed Bomb (Speed <˝ max speed to detonate)
N.B. This can be used with stationary vehicles to blow them up instantly!

PARK
----

PARK <DOOR OBJECT> 0 <DUMMY OBJECT> <DIRECTION> <SCORE>
DOOR OBJECT - normal roller DOOR feature, can be inside or outside garage.
DUMMY OBJECT - DUMMY used to decide actual block the garage is, this means you could park at the end of a long tunnel with the door at the start.
DIRECTION - not the same as directions for DOOR objects!  It tells GTA which side of the DUMMY you should respawn at.  Like this:
  2
1 ? 3
  0
SCORE - points to add.

EXAMPLE 1: Code Demos (Cerbera)
[1]

354 (21,22,3) DOOR 2 4 0 {The door object facing North}
500 (18,18,3) TRIGGER 2100 1
501 (27,20,4) FUTURECAR 22 768
502 (21,22,3) DUMMY 0 0

-1

2100 DISABLE 500 0 0 0 0 {Disable trigger}
2102 ARROW 502 0 0 0 0 {Point arrow to DUMMY location of garage}
2110 PARK 354 0 502 2 250 {Wait for player to park ANY car in garage}
